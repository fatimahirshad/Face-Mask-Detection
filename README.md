
# Face Mask Detection using YOLOv8

This project implements a real-time face mask detection system using YOLOv8 and computer vision.

## Features

- Detects whether a person is wearing a mask or not
- Real-time webcam detection
- Image inference
- Deep learning object detection using YOLOv8

## Dataset

Face Mask Detection Dataset containing images of:

- With Mask
- Without Mask

## Model

Model used: YOLOv8

Classes:
- with_mask
- without_mask

## Training Results

Precision: 0.994  
Recall: 0.996  
mAP50: 0.994
